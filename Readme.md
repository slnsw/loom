# SL - Loom

Install project dependencies:
```
$ npm install
$ bower install
```

Please add the GSAP/ColorPropsPlugin on index.html

```
<script src="/bower_components/gsap/src/uncompressed/plugins/ColorPropsPlugin.js"></script>
```


Run local dev server
```
$ gulp serve
```

Build the project (dist folder)
```
$ gulp
```
